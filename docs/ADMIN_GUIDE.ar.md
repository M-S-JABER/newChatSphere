# دليل المسؤول

هذا الدليل يشرح صلاحيات المسؤول فقط.

## إدارة المستخدمين
- إنشاء المستخدمين وتحديد الأدوار.
- تعديل اسم المستخدم أو الدور.
- إعادة تعيين كلمات المرور.
- حذف المستخدمين (لا يمكنك حذف حسابك).

## إعدادات حساب واتساب
- ضبط حساب واتساب الافتراضي.
- إدخال رقم الهاتف، رمز الوصول، ورمز تحقق الـ webhook.
- مراجعة حالة الحساب من صفحة الإعدادات.

## الرسائل الجاهزة
- إنشاء ردود جاهزة مسبقة.
- تفعيل أو تعطيل الرسائل دون حذفها.

## تشخيص الـ Webhook
- مراجعة أحداث الـ webhook.
- مسح سجلات أحداث الـ webhook.
- التحقق من حالة وإعدادات الـ webhook.

## الإحصائيات
- عرض حجم الرسائل ومؤشرات النشاط.
- مراجعة أداء الفريق ومؤشرات الاستجابة.

## سجلات المكالمات (للمسؤول)
- قائمة سجلات المكالمات متاحة من قائمة المستخدم.
- يتم حفظ سجلات المكالمات في localStorage داخل المتصفح افتراضيًا.

## تحكمات الـ API
- تحكمات واجهة API الخاصة بالمسؤول وإعدادات الـ webhook متاحة تحت المسار `/api/admin`.
