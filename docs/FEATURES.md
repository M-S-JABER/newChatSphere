# Features

This document lists the primary features in the product.

## Messaging
- Real-time message updates via WebSocket.
- Reply-to threading.
- Inbound and outbound media handling.
- Message search in a conversation.

## Conversations
- Archive and unarchive chats.
- Pin up to 10 conversations.
- Conversation info drawer with profile data and media lists.

## Templates and Ready Messages
- Template sending (Meta-approved templates).
- Ready messages configured by admins.

## Admin and Settings
- User management (admin only).
- WhatsApp instance configuration.
- Webhook diagnostics and testing.
- Statistics dashboard.

## Calling UX (UI Shell)
- Call overlay for incoming and outgoing calls.
- Call logs per conversation and admin global view.
- Note: calling is a UI layer only until Meta Calling is integrated.
