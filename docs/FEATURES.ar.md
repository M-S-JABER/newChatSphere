# الميزات

هذا المستند يعرض الميزات الأساسية في المنتج.

## الرسائل
- تحديثات فورية عبر WebSocket.
- ردود مرتبطة (Reply-to).
- معالجة الوسائط الواردة والصادرة.
- البحث داخل محادثة.

## المحادثات
- أرشفة وإلغاء الأرشفة.
- تثبيت حتى 10 محادثات.
- لوحة معلومات المحادثة مع بيانات الملف الشخصي وقوائم الوسائط.

## التمبلت والرسائل الجاهزة
- إرسال التمبلت (القوالب المعتمدة من Meta).
- رسائل جاهزة يحددها المسؤول.

## الإدارة والإعدادات
- إدارة المستخدمين (للمسؤول فقط).
- إعداد حساب واتساب.
- تشخيص واختبار الويبهوك.
- لوحة الإحصائيات.

## تجربة المكالمات (واجهة فقط)
- واجهة اتصال للمكالمات الواردة والصادرة.
- سجلات مكالمات لكل محادثة وعرض شامل للمسؤول.
- ملاحظة: المكالمات مجرد واجهة حتى يتم دمج Meta Calling.
