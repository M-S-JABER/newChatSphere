# Admin Guide

This guide covers admin-only capabilities.

## User Management
- Create users and assign roles.
- Edit username or role.
- Reset passwords.
- Delete users (cannot delete your own account).

## WhatsApp Instance Settings
- Configure the default WhatsApp instance.
- Provide phone number id, access token, and webhook verify token.
- Review instance status in Settings.

## Ready Messages
- Create predefined responses.
- Enable or disable without deleting.

## Webhook Diagnostics
- Review webhook events.
- Clear webhook event logs.
- Check webhook status and configuration.

## Statistics
- View message volume and activity metrics.
- Review team performance and response metrics.

## Call Logs (Admin)
- Global call log list is available from the user menu.
- Call logs are stored in the browser localStorage by default.

## API Controls
- Admin API controls and webhook config are exposed under /api/admin.
