# نظرة عامة على الوسائط

معالجة الوسائط موزعة بين التنزيل الوارد، المعالجة، والتسليم بروابط موقّعة.

## المفاهيم الأساسية
- الوسائط الواردة تُنزّل من Meta وتُخزّن تحت uploads/.
- يتم إنشاء صور مصغرة للمعاينات عند الحاجة.
- روابط الوسائط تُخدم عبر طبقة روابط موقّعة.
