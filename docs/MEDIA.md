# Media Overview

Media handling is split across inbound downloads, processing, and signed delivery.

## Core Concepts
- Inbound media is downloaded from Meta and stored under uploads/.
- Thumbnails are generated for previews where applicable.
- Media URLs are served through a signed URL layer.
