# User Guide

This guide describes common actions for agents and standard users.

## Login
1. Open the app and sign in with your username and password.
2. You will land on the chat list.

## Start a Conversation
1. Click "New conversation".
2. Enter the customer phone number in international format.
3. Send an initial message if needed.

## Send a Message
1. Type in the composer.
2. Press Enter to send.
3. Use Shift+Enter for a newline.

## Send Media
1. Click the attachment icon.
2. Select one or more files.
3. Wait for upload completion.

## Reply to a Message
1. Hover a message and choose Reply.
2. Type and send your response.

## Search Messages
1. Click the search icon in the thread header.
2. Type your query and navigate matches.

## View Chat Info
1. Click the info icon to open Chat info.
2. Review media, files, links, and call history.

## Calls
The call UI shows incoming and outgoing call states.
Calls are currently a UI shell and do not place real Meta calls.
