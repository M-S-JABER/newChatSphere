# دليل التشغيل

## السجلات
- سجلات HTTP تُكتب عبر pino.
- اضبط LOG_LEVEL للتحكم في مستوى التفاصيل.

## النسخ الاحتياطي
- انسخ Postgres بانتظام (pg_dump أو نسخ مُدارة).
- انسخ مجلد uploads إذا كنت تخزن الوسائط محليًا.

## التخزين
- مجلد uploads/ قد ينمو بسرعة؛ راقب مساحة القرص.
- ضع في الاعتبار التخزين الخارجي إذا كان حجم الوسائط كبيرًا.

## التحديثات
- نفّذ تزامن قاعدة البيانات بـ `npm run db:push`.
- أعد البناء والتشغيل بعد تغييرات الكود.

## المراقبة
- تابع أزمنة استجابة الـ API وسجلات الأخطاء.
- راقب نجاح تسليم الويبهوك عبر واجهة التشخيص.
